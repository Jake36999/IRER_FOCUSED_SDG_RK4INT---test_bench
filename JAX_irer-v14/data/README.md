# IRER V14 Data Module

This directory contains resources related to the data management of the IRER V14 architecture. 

## PostgreSQL Migration

The `postgres_migration.sql` file includes the SQL commands necessary to migrate the database from SQLite to PostgreSQL for the Knowledge Extraction Log (KEL). This migration is crucial for ensuring scalable storage and efficient data retrieval.

## Usage

Ensure that you execute the migration script in your PostgreSQL environment to set up the necessary database structure for the project. 

For any additional data-related functionalities or queries, refer to the respective modules and documentation within the project.
